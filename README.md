# Rails Forms

Created as part of The Odin Project.[^1]

## Requirement

- Ruby `3.2.2`
- Rails `7.1.3`
- sqlite3 `1.4`

## Install (First time only)

Dependencies

- Install ruby `3.2.2`
- Install rails dependencies: `bundle install`

## Database creation & initialization

- `bin/rails db:create`
- `bin/rails db:migrate`
